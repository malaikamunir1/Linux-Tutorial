Linux Tutorial

Learn basic Linux commands and concepts quickly!

1. What is Linux?

Linux is an open-source operating system used on servers, desktops, and embedded systems. It's known for stability, security, and flexibility.

2. Navigating Directories

cd (Change Directory) lets you move between folders. Example: cd Documents.

3. Listing Files

ls lists files and folders in a directory. Use ls -l for detailed view.

4. Creating Files and Folders

touch creates a file (e.g., touch file.txt). mkdir creates a directory (e.g., mkdir new_folder).

5. Deleting Files and Folders

rm deletes files (rm file.txt). Use rm -r folder/ to delete folders.

6. Viewing File Content

cat displays file contents. Example: cat file.txt. Use nano or vim for editing.

7. File Permissions

chmod changes permissions. Example: chmod 755 file.sh gives execute permission.

8. User Management

adduser adds a new user. passwd changes a user password.

9. Package Management

apt (Debian/Ubuntu) installs software: sudo apt install package-name.

10. Basic Networking

ping checks connection: ping google.com. ifconfig or ip addr shows network settings.
